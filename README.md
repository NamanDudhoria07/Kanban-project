# 📝 Collaborative Kanban Board

### A sleek, interactive Kanban Board designed for seamless task management across devices. Featuring a glassmorphic nebula interface, real-time priority sorting, and a custom-engineered mobile drag-and-drop system that balances smooth scrolling with intentional task movement.

---

## 📌 Problem Statement
<font size="3">To build a **functional Kanban Board** that:</font>
* Enables users to **create, edit, and delete** tasks dynamically.
* Automatically **sorts tasks** by High, Medium, and Low priority.
* Provides a **Progress Tracker** that updates in real-time as tasks move to "Done."

---

## 🛠️ Tech Stack Used
* **HTML5** & **CSS3** (Glassmorphism UI)
* **JavaScript** (ES6+)
* **React.js** (State Management & Hooks)
* **Netlify** (Continuous Deployment)
